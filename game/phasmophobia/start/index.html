# 开始

你将会阅读到大量的有关《Phasmophobia》（恐鬼症）的游戏攻略，该攻略的内容存在自制与转载混合发布，其中所有的内容遵循CC BY-NC协议。将会在本文档末尾详细说明该协议，以及你在转载文档时所需要的限制。

# 开始成为一名合格的鬼魂调查员

在游戏中，你将在一张地图里寻找一个鬼，鬼的类型十分丰富，且每种鬼都有不同的特性。在调查过程中，你需要用你卡车内的道具来找到鬼并分辨出鬼的类型。其中鬼有可能会尝试把你杀了，你需要在不被鬼杀死的前提下根据鬼留下的线索与证据判断鬼的类型，然后开溜！

# 游戏详细流程

选择一张地图，通常是一座房子。

鬼会在房子内的其中一个房间内，当然在有些情况下，鬼会到处乱走。

根据你车里有的道具，看看鬼会做出什么反应。比如鬼开门的时候留下的手掌印，你可以拿出你的紫外线手电筒找找看有没有手印。

把灵异事件都用相机拍下来。

根据你遇见的所有灵异事件，判断是什么鬼。

你可以到处走走这座房子，一般来说都会有一个诅咒道具 以及一块骨头。骨头是一个好证据，拍下来然后拿走；诅咒道具可能会对你找鬼有帮助。（或者有反帮助）

但请确保你的理智不要太低，要不然鬼会直接开始猎杀。

如果鬼猎杀了，请找地方躲好（或者找个柱子绕）

继续判断鬼的特点，确定鬼的类型。

当你觉得你找到了足够的证据、确定了鬼的类型的时候，抓紧跑。

回到你的卡车上，把卡车门关上就可以了！

道具？自己会飞回来的！（只要没人死就好）

鬼的类型被你判断正确的时候，游戏会有很多奖金。拿奖金去买更多的道具，或者升级更好的道具，让自己下次找鬼的时候更如鱼得水。
