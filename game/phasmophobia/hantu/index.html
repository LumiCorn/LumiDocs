    御灵

    # 👻恶魔Hantu

    • 更早发起猎杀
    • 猎杀条件不受 理智值 限制

    弱点：十字架更有效（生效距离会是十字架原先的150%）

    # 特性

    恶魔的猎杀CD时间（即两次猎杀之间最短的冷却时间）为20秒，这比一般鬼魂要早5秒。

    熏香驱散恶魔的时间为60秒，这比一般鬼魂要早30秒。

    恶魔通常在平均理智为70%就开始猎杀，这比一般鬼魂要高20%。

    十字架的生效范围是原先的150%，这比一般鬼魂要扩大50%。

    # 技能

    猎杀间隔CD下降到20秒

    # 猎杀条件

    • 平均理智值<70%
    • 有可能无视理智值就开始猎杀
