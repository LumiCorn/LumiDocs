    赤鬼

    # 👻赤鬼Oni

    • 附近玩家较多时活动频率增加

    弱点： 不会哈气；猎杀时闪烁隐身时间变短

    # 特征

    玩家较多时赤鬼将更频繁地与物体互动。

    对于鬼魂事件，赤鬼不会产生烟雾型哈气。但会有其他哈气。

    现身状态下，全现身的几率会比黑影、半透明、影子的几率高。

    鬼魂事件状态下，目标玩家与赤鬼产生碰撞会降低20%理智值，而其他的鬼是10%。

    赤鬼会阻止部分地图生成大型烟雾，详见“鬼魂事件”定义中对烟雾的解釋。
